# Inventory Management System

A Java-based desktop application for managing inventory items. This project is built using Java Swing and is designed to perform basic CRUD operations for inventory tracking.

---

## 📌 Features

- Add new products/items
- Update existing item details
- Delete items from inventory
- View all stored items
- Simple and user-friendly GUI
- Data handling using Java file handling (or database if you used it)

---

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- File Handling
- Eclipse IDE
